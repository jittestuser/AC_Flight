# Test Scenarios for Air Canada Automation (US 001)

## Scenario 1: Launch Air Canada Portal
- Open https://www.aircanada.com/
- Verify the homepage loads successfully

## Scenario 2: Access Flight Status Tab
- Click on the “Flight status” tab
- Verify the flight status page is displayed

## Scenario 3: Search Flight Status by Flight Number (Past 3 Days)
- Enter flight number: AC 792
- Select a date within the past 3 days
- Click search
- Verify flight status is displayed for the selected date

## Scenario 4: Search Flight Status by Flight Number (Future Days)
- Enter flight number: AC 792
- Select a date within a few days in the future
- Click search
- Verify flight status is displayed for the selected date

## Scenario 5: Search Flight Status by Route (YYZ - YHZ)
- Select route: YYZ to YHZ
- Select a valid date
- Click search
- Verify flight status is displayed for the selected route and date

## Scenario 6: Invalid Flight Number
- Enter an invalid flight number
- Click search
- Verify appropriate error message is displayed

## Scenario 7: No Results for Selected Date
- Enter flight number: AC 792
- Select a date with no flights
- Click search
- Verify “No results found” message is displayed
