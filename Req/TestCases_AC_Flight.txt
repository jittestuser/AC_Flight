# Test Cases for Air Canada Automation (US 001)

## Test Case 1: Verify Air Canada Portal Launch
- Step: Open https://www.aircanada.com/
- Expected Result: Homepage loads successfully

## Test Case 2: Verify Flight Status Tab Access
- Step: Click on “Flight status” tab
- Expected Result: Flight status page is displayed

## Test Case 3: Search Flight Status by Flight Number (Past 3 Days)
- Step 1: Enter flight number: AC 792
- Step 2: Select a date within the past 3 days
- Step 3: Click search
- Expected Result: Flight status for AC 792 is displayed for the selected date

## Test Case 4: Search Flight Status by Flight Number (Future Days)
- Step 1: Enter flight number: AC 792
- Step 2: Select a date within a few days in the future
- Step 3: Click search
- Expected Result: Flight status for AC 792 is displayed for the selected date

## Test Case 5: Search Flight Status by Route (YYZ - YHZ)
- Step 1: Select route: YYZ to YHZ
- Step 2: Select a valid date
- Step 3: Click search
- Expected Result: Flight status for YYZ - YHZ is displayed for the selected date

## Test Case 6: Invalid Flight Number
- Step: Enter an invalid flight number and click search
- Expected Result: Appropriate error message is displayed

## Test Case 7: No Results for Selected Date
- Step 1: Enter flight number: AC 792
- Step 2: Select a date with no flights
- Step 3: Click search
- Expected Result: “No results found” message is displayed
